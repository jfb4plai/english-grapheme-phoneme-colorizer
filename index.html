How It Works

1

Install the Add-in

Download the manifest file and install it in Microsoft Word through the Developer tab - takes less than 2 minutes!

2

Select Text

Highlight any text in your Word document - from single words to entire pages. The add-in works with any English text.

3

Click Colorize

Use the "Quick Colorize" button in the new Dyslexia Support tab to instantly apply grapheme-phoneme color patterns.

4

Read with Confidence

Students can now see phonetic patterns visually, making it easier to decode unfamiliar words and build reading fluency.
